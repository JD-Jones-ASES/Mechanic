# ADR-0005: Licensing — MIT code, CC BY 4.0 content & dataset

**Status:** accepted (default; reversible until the repo goes public)

**Decision.** Code (pipeline, engines, components) is MIT (`LICENSE`). Prose, derivations, figures, and the
curated materials dataset are CC BY 4.0 (`LICENSE-content.md`). The site carries an attributions/credits
page (citing MIL-HDBK-5J, ASTM designations, USDA, USGS, CDA, vendor TDS sources) and a prominent
"educational — not for design use" disclaimer.

**Why.** CC BY lets other educators reuse the material with attribution and is compatible with every data
source we use. Policy line (also in `docs/data-provenance.md`): no CC BY-NC-SA and no MMPDS-derived
material may enter the content pipeline — NC/SA terms would contaminate reuse, and MMPDS is copyrighted
with strict copy limits.
